1. Latar Belakang Shopee

Shopee merupakan ekosistem e-commerce masif yang membutuhkan manajemen data yang sangat kompleks. Pendekatan pemrograman prosedural tidak lagi efisien untuk mengelola jutaan interaksi produk dan user. Paradigma OOP diterapkan untuk menciptakan sistem yang modular, aman, dan mudah dikembangkan di masa depan.

2. Masalah yang Sering Dihadapi

  • Sinkronisasi Stok: Data stok tidak sinkron secara real-time saat terjadi transaksi massal (misal: flash sale).
  • Kompleksitas Skalabilitas: Penambahan kategori produk baru sering menyebabkan sistem menjadi tidak stabil.
  • Integritas Data: Risiko manipulasi data transaksi jika akses atribut tidak dibatasi (kurangnya enkapsulasi).

3. Mengapa OOP?

OOP mempermudah pengembangan dengan konsep Encapsulation (keamanan data), Inheritance (penggunaan kembali kode), dan Modularity (pemecahan sistem menjadi bagian-bagian kecil yang mudah dipelihara).

4. Langkah Penyelesaian

Analisis Entitas → Desain Class → Pemecahan File Dart → Integrasi Sistem → Pengujian.

Source Code 1: product.dart (Abstraksi)

abstract class Product {
  String name;
  double price;
  Product(this.name, this.price);
  void displayInfo();
}

Source Code 2: electronic.dart (Inheritance)

import 'product.dart';
class Electronic extends Product {
  Electronic(String name, double price) : super(name, price);
  @override
  void displayInfo() => print("Elektronik: $name - Rp$price");
}

Source Code 3: cart.dart (Composition)

import 'product.dart';
class Cart {
  List items = [];
  void addItem(Product p) => items.add(p);
  double getTotal() => items.fold(0, (s, i) => s + i.price);
}

Source Code 4: user.dart (Encapsulation)

import 'cart.dart';
class User {
  String _name;
  Cart cart = Cart();
  User(this._name);
  String get name => _name;
}

Source Code 5: main.dart (Eksekusi)

import 'user.dart';
import 'electronic.dart';

void main() {
  var user = User("Serlin & Jeinnet");
  user.cart.addItem(Electronic("Smartphone", 5000000));
  print("Total Pembayaran: Rp${user.cart.getTotal()}");
}
